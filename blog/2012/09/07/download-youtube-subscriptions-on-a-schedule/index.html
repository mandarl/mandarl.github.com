Download Youtube Subscriptions on a Schedule

Sep 7th, 2012

My BSNL broadband plan gives me unlimited data between 2 AM and 8 AM. I thought this would be a great time to download all my YouTube subscriptions. I could watch the videos the next day without using my download quota! I found a couple of great scripts to get the job done. I had to make a few changes to get them working on windows:

Prerequisites: 1. Cygwin or another shell for windows. 2. Python

Just run the subscriptions script like so:

 ./download-youtube-subscriptions.sh mandarlimaye

Put this command in a batch file and schedule it to run whenever you like.
